# Vietnamese-English Code-Switching Speech Translation

This project aims to build an End-to-End (E2E) Speech Translation model for Vietnamese-English Code-Switching data. The system processes code-switched audio (e.g., from YouTube or blogs), transcribes it, and translates it into Vietnamese.

## Project Structure

- `dataset/`: Contains audio files and the SQLite database.
- `src/`: Source code for preprocessing, training, and utilities.
- `docs/`: Documentation for the project workflow.
