## Why Archived?

The project migrated from Streamlit to NiceGUI for the following reasons:

1. **Better event-driven architecture** - NiceGUI's reactive model is cleaner
2. **Keyboard shortcuts** - NiceGUI supports proper keyboard event handling
3. **Modern UI** - NiceGUI offers more flexibility and better UX
4. **Performance** - NiceGUI has better performance for real-time updates

This implementation was functional and served well during early development. Archived for historical reference and potential feature comparison.
